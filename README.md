# Telangana Growth Analysis Dashboard

This project analyzes Telangana's economic growth using public datasets from the Telangana Open Data Portal.

## Tools Used
- Power BI
- DAX
- Data Modeling

## Key Analysis Areas
- Stamp Registration Revenue
- Vehicle Sales Trends
- Industrial Investments (TS-iPASS)

## Key Insights
- Stamp revenue increased significantly between FY2019 and FY2022.
- Electric vehicle adoption increased rapidly in Rangareddy district.
- Vehicle sales peak during the Oct–Dec festive season.
- Rangareddy leads in commercial property investment.
